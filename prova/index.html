JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Estruturas condicionais

São instruções para realizar determinadas tarefas a partir de uma condição, seja de decisão ou repetição; Ex.: Um jogo precisa mudar o placar toda vez que um jogador marca pontuação;

Arrays e objetos

Arrays são um tipo de lista, ou matriz de variáveis, onde cada variável possui um índice. Os valores podem ser de vários tipos. Imagina que um array é uma caixa com várias outras caixas dentro e cada uma contendo algum valor; Ex.: let array = [‘string’, 1, true…]


Objetos são dados que possuem propriedades e valores que definem suas características. Deve ser declarado entre chaves “{}”. Ex.: imagine uma xícara azul. Ela tem cor, pode ter vários tamanhos e funções(quero cafééé). Pode ser declarada assim: var xicara = { cor: ‘azul’, tamanho: ‘p’, funcao: tomarCafe() }
